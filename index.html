Reflow Maven Skin

Responsive Apache Maven skin to reflow the standard Maven site with a modern feel

Get Started

Reflow Maven skin

is an Apache Maven site skin built on Bootstrap. It allows various structural and stylistic customizations to create a modern-looking Maven-generated website.

Responsive Layout

Write your pages in APT, AsciiDoc, Xdoc, Xhtml or Markdown, then restructure them using Reflow skin layouts.

Reflow skin is responsive thanks to Bootstrap, so the layouts will be rearranged automatically for readability on small screens.

Read more »

Theme

The skin theme can be switched easily: just select a Bootstrap theme to give an easy makeover for your Maven site.

  • Default - use the default Bootstrap theme
  • Bootswatch - select an excellent free theme from Bootswatch
  • Custom - create your own Bootstrap theme with existing tools

Need to change something? Extend the skin with custom site.css file in your project, and reuse it for multi-module site.

How to select a theme »

Configure

Many features and components are configurable to enhance your web site.

  • Table of contents - display ToC for each page: top bar or sidebar
  • Menus - filter Maven menus and select what to display in top or bottom navigation
  • Code highlight - syntax colouring for code snippets
  • Image preview - display images in pop-ups
  • Documentation support - organize easier your multi-section documentation.
