Layout Overview

This section describes how apply different layouts at your maven site.

  • Page Layout is the standard representation in Reflow skin.
  • Document Layout allows to organize your documentation in single page representation.
  • Body Layout allows to be free in your inspiration to create a attractive page.
  • Section Layout allows customizing the page layout and reflowing the page using columns, thumbnails, or with the carousel component.
