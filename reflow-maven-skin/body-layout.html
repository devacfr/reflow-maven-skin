Page Body layout

Body layout is composed only header (navbar), body and footer. You can play only with the body content. The content is put as is, Reflow maven skin should not modify this content. Letâ€™s be free in your inspiration to create a attractive home page.

Usage

support v2.3 new
element reflow element

To declare a page as body layout, create an page (index.html in example below) and set type attribute with value body.

<pages>
    ...
    <index type="body" project="project-id">
    </index>
    ...
</pages>
  • project v1.0 - project attribute associates the page to specific Maven artifactId declaring in the project. This can be useful to exclude all inherit configuration on same page name localized in other module in project. Be careful that name attribute in root <project> element in site.xml is used to set the site title.
