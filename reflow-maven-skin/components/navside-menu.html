Navside Menu Component

Navside menu component is a part of page documentation layout to the left of page. Navside menu is vertical multi level menus for both desktop and mobile modes initialized with the <menu> element declared in a page with documentation layout type.

Usage

support v2.0 new
element reflow component

<navside-menu cssClass="bootstrap classes"
              theme="light|dark"
              background="bootstrap colour theme" />
  • cssClass v2.0 - Sets the css class with built-in Bootstrap support. navside-light bg-light is the default value if value is empty or null. note: cssClass override attributes theme and background if is filled.
  • theme v2.0 - Sets the css class with built-in Bootstrap theme . light is the default value if value is empty or null.
  • background v2.0 - Sets the css class with built-in Bootstrap colour theme. light is the default value if value is empty or null.
